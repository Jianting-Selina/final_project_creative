Main Content: A card-based feed of events. Each card displays: Event Title, Organization Logo, Date/Time, Distance (e.g., "1.2 km away"), and tags (e.g., "Education," "Outdoor").

12.2 Screen 2: The "Active Shift" (Geofence Triggered)

Context: This screen appears when the user taps the "You have arrived" notification.
Visuals: A map view centered on the user's location relative to the Event Geofence (visual confirmation).
Status Indicator: "You are at: Riverdale Food Bank."
Input Control: A large "Slide to Check In" slider at the bottom (prevents accidental clicks).
Fallback: A small link text "GPS not working? Scan QR Code" below the slider.

12.3 Screen 3: The "Certificate Generator"

Context: Accessed via Profile -> My Records.
Input Controls:
Date Range Picker: "Start Date" to "End Date".
Organization Filter: Dropdown to select specific orgs or "All".
Format Selector: Radio buttons for "Ontario High School Diploma (40 Hours)" or "Professional Reference Portfolio".
Options: Checkbox for "Include Supervisor Contact Info" (Required for references).
Action: "Generate PDF" button.
Output: A preview of the PDF with a "Download/Share" floating action button.
